## Variant: Clean Editorial (Web Admin Submissions)

### Design stance
Notion/Linear/Cal.com 스타일의 테이블 중심 에디토리얼 레이아웃. 넓은 화면을 활용한 데이터 테이블, 사이드바 네비게이션, 모달 상세 보기, 페이지네이션, 검색/필터 툴바. 데스크톱 관리자 업무에 최적화된 정보 밀도와 가독성.

### Key choices
- **Layout**: 고정 사이드바(280px) + 데이터 테이블, 모달 상세/편집, 반응형 가로 스크롤
- **Typography**: DM Sans, 32px 페이지 제목, 14px 테이블 본문, 12px 헤더(대문자)
- **Color**: 파란색 액센트(#0066ff), 연한 회색 배경, 시맨틱 상태 배지, 모달 오버레이
- **Interaction**: 행 클릭/키보드 모달 열기, 체크박스 다중 선택, 페이지네이션, 빠른 반려 사유 칩, 검색 디바운스
- **Spacing**: 20px 테이블 패딩, 32px 페이지 여백, 24px 모달 패딩

### Trade-offs
- **Strong at**: 대량 데이터 스캔, 키보드 중심 워크플로, 정렬/필터/검색 조합, 전문적 관리자 도구 느낌
- **Weak at**: 모바일에서 테이블 가로 스크롤 필요, 시각적 계층이 카드형보다 약함

### Best for
- 일일 50+건 처리하는 운영팀, 키보드 파워유저, 데이터 분석/익스포트 필요 시, 데스크톱 전용 어드민
