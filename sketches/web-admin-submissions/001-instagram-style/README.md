## Variant: Instagram Style (Web Admin Submissions)

### Design stance
Instagram/Threads 웹 버전과 네이티브 앱의 디자인 언어를 데스크톱 어드민에 맞게 확장한 변형. 사이드바 네비게이션, 카드 기반 리스트, 상태 배지, 확장형 상세 패널, 모달 반려 사유 입력. 기존 웹 어드민 구현(`SubmissionCard`)을 디자인 시스템 토큰으로 리팩토링.

### Key choices
- **Layout**: 고정 사이드바(260px) + 메인 콘텐츠, 반응형(모바일에서 사이드바 드로어)
- **Typography**: Geist, 28px 페이지 제목, 16px 카드 제목, 14px 본문, 11px 라벨
- **Color**: 파란색 프라이머리, 시맨틱 상태 색상(대기/승인/반려/검수/중복), 연한 배경으로 확장 상태 표시
- **Interaction**: 카드 클릭 확장, 인라인 편집 폼, 승인/반려/저장 액션, 빠른 반려 사유 버튼, 토스트 알림
- **Spacing**: 16px 기본, 24px 섹션, 12px 카드 내부

### Trade-offs
- **Strong at**: 모바일 앱과 시각적 일관성, 직관적 카드 인터페이스, 상태별 색상 구분 명확
- **Weak at**: 대량 데이터(100+)에서 스크롤 길어짐, 테이블 뷰 대비 정보 밀도 낮음

### Best for
- 기존 모바일 앱 사용자가 웹 어드민 진입 시 이질감 없는 경험, 중소규모 제보 관리(~50건/일)
