## Variant: Instagram Style (Mobile Submit)

### Design stance
Instagram/Threads 네이티브 앱의 디자인 언어를 충실히 따르는 변형. 라운드 카드, 미묘한 그림자, 파란색 프라이머리, 8px 그리드 시스템, 시스템 폰트(Geist) 사용. 사용자가 이미 익숙한 소셜 앱 UX를 그대로 제공.

### Key choices
- **Layout**: 단일 컬럼 스크롤, 카드 기반 폼 그룹핑
- **Typography**: Geist 시스템 폰트, 28px 헤드라인, 15px 본문, 13px 라벨
- **Color**: 파란색(#4f63d7) 프라이머리, 성공/경고/에러 시맨틱 색상, 표면/배경/보더 분리
- **Interaction**: 실시간 인라인 검증, 토스트 알림, 로딩 상태, 키보드 네비게이션
- **Spacing**: 16px 기본 여백, 20px 섹션 간격, 12px 필드 간격

### Trade-offs
- **Strong at**: 사용자 친숙도, 접근성(시맨틱 색상, 라벨 연결), 반응형(모바일 퍼스트)
- **Weak at**: 정보 밀도(긴 폼에서 스크롤 필요), 시각적 차별화(인스타그램과 유사)

### Best for
- 일반 사용자 대상 제보 플로우, 모바일 앱/웹 공통 사용, 빠른 온보딩이 중요한 경우
